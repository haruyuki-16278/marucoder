# 教員向け運用手順 (MVP)

最終更新日: 2026-06-25
対象: 授業担当教員

## 1. 授業前準備

1. 教卓PCでブラウザを開き、教卓進捗画面にアクセスする。
2. 授業用グループを作成する。
3. 学生へ参加コードを配布し、参加完了を確認する。
4. 席順を登録する。
5. 課題IDを入力し、表示対象を授業課題に合わせる。

## 2. 席順登録方法

### 2.1 手動登録

1. 教卓進捗画面の「席順編集」から row / col / userId / label を入力する。
2. 席更新を押す。
3. マップ表示に反映されることを確認する。

### 2.2 CSV登録

1. 以下フォーマットでCSVを作成する。

```csv
groupId,row,col,userId,studentName,label
group_a,1,1,s001,山田 太郎,01
group_a,1,2,s002,佐藤 花子,02
```

2. CSV取込欄に貼り付け、CSV取込を実行する。
3. エラー表示がある場合は line と reason をもとにCSVを修正して再実行する。

## 3. 授業中の確認

1. グループ進捗テーブルで進捗率を確認する。
2. 席順マップで詰まっている学生 (WA/CE/RE/TLE) を確認する。
3. 席セルをクリックして提出履歴へ遷移し、詳細を確認する。
4. 課題切替で別課題の進捗へ切り替える。

## 4. 授業後の確認

1. 提出履歴画面で課題ごとの提出状況を確認する。
2. 必要に応じてCSVを再投入し、次回授業に備えて席順を調整する。

## 5. 注意事項

1. 教卓表示は10秒ごとに自動更新される。
2. 空席は userId 未設定で管理する。
3. 学生名表示は氏名表示ポリシーに従う。
